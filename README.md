## 🧠 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Bheem-Works/VimIndexPortfolio.git
   cd portfolio-website
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html
   ```

3. Customize the content, images, and links to make it your own.
